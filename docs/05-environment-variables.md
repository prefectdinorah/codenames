# Переменные окружения

Источник правды в репозитории: **`.env.example`**. Файл **`.env`** создаётся локально / на сервере и **не коммитится**.

## Spyfall — картинки локаций

| Переменная | Назначение |
|------------|------------|
| `SPYFALL_IMAGE_BASE` | Публичный базовый URL каталога с файлами картинок (без завершающего `/` допускается гибко — см. код). |
| `SPYFALL_IMAGE_EXT` | Расширение файла, по умолчанию `jpg`. |

Если база пустая, клиент рисует заглушки.

## Monopoly — публичные URL логотипов

| Переменная | Назначение |
|------------|------------|
| `MONOPOLY_IMAGE_BASE` | Базовый URL для отображения загруженных логотипов. |
| `MONOPOLY_IMAGE_EXT` | По умолчанию `png`. |

## S3 (Timeweb и совместимые)

Используются **сервером** (сохранение `monopoly/state.json`, заливка логотипов) и скриптом **`scripts/upload-spyfall-images.sh`**.

| Переменная | Назначение |
|------------|------------|
| `S3_ENDPOINT` | URL эндпоинта S3-совместимого API. |
| `S3_REGION` | Регион для SDK. |
| `S3_BUCKET` | Имя бакета. |
| `S3_PREFIX` | Префикс для скрипта Spyfall (сервер монополии использует ключи `monopoly/...` в коде `monopoly-store.js`). |
| `S3_ACCESS_KEY` / `S3_SECRET_KEY` | Ключи. **Пустые** → режим без S3, состояние монополии только в памяти. |

## Обслуживание

| Переменная | Назначение |
|------------|------------|
| `MAINTENANCE_MODE` | `1` — включён режим заглушки для всех без cookie. |
| `MAINTENANCE_PASS` | Секрет для `?bypass=...` в query; выставляет HttpOnly cookie на 30 дней. |

## Порт

| Переменная | Назначение |
|------------|------------|
| `PORT` | Порт HTTP/WS; в Docker обычно 3000. |
