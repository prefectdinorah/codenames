# Режимы игр (логика на сервере)

Все режимы делят общие сообщения: **`create-room`**, **`join-room`**, **`change-name`**, **`pick-team`**, **`toggle-pause`**. Дальше — специфичные обработчики в `server.js`.

## Codenames (`gameMode === 'codenames'`)

- Настройки: число команд, размер сетки, таймер хода.
- Роли: `spymaster` / `operative`, голоса оперативников, подтверждение карточки с задержкой.
- Слова из `words.js`.

## Alias

- Фазы: ожидание → объяснение → разбор очков → следующая команда.
- Слова: `alias-words.js`.

## Spyfall

- Фазы: лобби → игра → голосование по обвинению → финал.
- Локации: `spyfall-locations.js`, картинки через env.

## Крокодил (`crocodile`)

- Рисование: сообщения **`croc-draw`**, **`croc-clear`** идут **напрямую** другим сокетам, не через полный `state` (меньше трафика).

## Кто я? (`whoami`)

- Назначение слов в фазе настройки, блокнот без широкого broadcast (см. `save-notebook`).
